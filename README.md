## 📌 Project Overview

The **Peter Harvard INT'L School Alumni Platform** is a web application designed to connect graduates with their alma mater through a centralized digital community.

Alumni can register, access a directory, and view school announcements, while school administrators can manage members, highlight featured alumni, and post events. The goal is to strengthen long-term relationships between the school and its alumni while delivering a modern, user-friendly experience.

---

## 🎯 Project Objectives

- Create a structured, searchable alumni database
- Highlight successful alumni and their achievements
- Enable easy communication for announcements and events
- Build a scalable, maintainable, and visually polished web interface

---

## ✨ Core Features (MVP)

| Feature | Description |
|---|---|
| 📝 Alumni Registration Form | Alumni can sign up and create their profile |
| ✅ Admin Approval System | Admins review and approve/reject registrations |
| 🔍 Alumni Directory | Searchable by name and graduation year |
| 🌟 Featured Alumni Section | Showcase notable graduates and achievements |
| 📅 Events & Announcements | Admin-managed posts for school updates |
| 🛠️ Admin Dashboard | Central control panel for all admin operations |

## 🛠️ Tech Stack

### Frontend
- **React** — Component-based UI framework
- **Tailwind CSS** — Utility-first styling
- **Context API** — Global state management

### Backend / Database
- **Supabase** — Authentication, database, and admin management

### Version Control
- **Git & GitHub**

---

## 👥 User Roles

### 🔐 Admin
- Manage alumni registrations
- Approve or reject accounts
- Feature alumni profiles
- Post events and announcements

### 🎓 Alumni
- Register and create a profile
- Access the alumni directory
- View featured alumni, events, and updates
